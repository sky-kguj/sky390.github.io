GitHub创建个人主页报错404 There is not a GitHub Pages site here.

目录

使用GitHub创建好个人主页后访问报错：404 There isn't a GitHub Pages site here.

一、问题原因

出现该问题的原因是创建的username/username.github.io项目中没有内容。

二、解决方案

2.1 快速解决方案

使用GitHub预定义的主题快速添加内容

1、进入setting

2、找到下图中箭头所指向的按钮

3、点击Change theme（未设置之前不知道是不是这个，设置之后即可访问）

（第四点来自评论区，感谢 AnowBC）

4.点击 change theme 之后底下有个link-to-another-pages ，然后再选用主题就好了，这样可以跳转到你用hexo部署的界面

2.2 根据需要添加个人主页内容

注：注意默认链接为index.html
